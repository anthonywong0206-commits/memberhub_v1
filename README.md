# MemberHub 靜態版 v3

更新內容：
- 會員管理列表只顯示「姓名」及「會員編號」
- 每位會員設有「查看」及「刪除」按鍵
- 查看會員詳情需要輸入系統密碼
- 刪除會員需要輸入系統密碼
- 如未設定密碼，系統會提示先到設定頁建立密碼

## 安裝 / 更新方法

1. 解壓 ZIP
2. 將以下檔案上載到 GitHub repo 根目錄並覆蓋舊檔案：
   - index.html
   - style.css
   - app.js
   - README.md
3. GitHub Pages 會自動更新
4. 如使用 Vercel，直接重新部署即可

不需要 npm，不需要 package.json，不需要 node_modules。
