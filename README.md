# MemberHub 靜態版

## 使用方法
1. 解壓 ZIP
2. 將 `index.html`、`style.css`、`app.js`、`README.md` 放到 GitHub repo 根目錄
3. GitHub Pages 設定：Settings → Pages → Deploy from branch → main → /(root)

## Vercel 使用方法
- Import GitHub repo
- Framework Preset: Other
- Build Command: 留空
- Output Directory: 留空

## 注意
- 不需要 npm install
- 不需要 package.json
- 不要上載 node_modules
- 所有會員資料只存在使用者瀏覽器 localStorage
